JavaScript Cheat Sheet

Concept Description
Variable Declaration: var, let, const Declares variables with different scopes and mutability
Functions: function myFunction() { } Declares a function with a specific set of instructions
Arrow Functions: const myFunction = () => { } Declares a function in a concise form that retains the context of 'this'
Conditions: if (condition) { } else { } Executes different code blocks based on a boolean condition
Loops: for (let i = 0; i < 10; i++) { } Repeats a block of code a certain number of times
Loops: while (condition) { } Repeats a block of code while a condition is true
Arrays: let arr = [1, 2, 3]; Declares an array to store multiple values in a single variable
Objects: let obj = { key: 'value' }; Declares an object to store keyed collections of various data
Event Handling: element.addEventListener('click', function); Attaches an event handler to an element that executes when the event occurs
DOM Manipulation: document.getElementById('id') Accesses and manipulates elements of the HTML document
Promises and Async/Await Handles asynchronous operations using a more manageable interface. Simplifies working with asynchronous promises.
Template Literals Allow strings to include embedded expressions and multi-line capabilities using backticks (\`\`).
Modules Enable modular programming by allowing import and export of functionality between JavaScript files.
Destructuring Assignment Allows unpacking of values from arrays or properties from objects directly into distinct variables.
Spread and Rest Operators Enables expanding or collecting elements with the spread (\`...\`) and rest (\`...\`) operators in function calls and array literals.
