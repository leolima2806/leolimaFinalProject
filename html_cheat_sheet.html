HTML Cheat Sheet

Tag Description
<h1> to <h6> Headers of different levels
<p> Defines a paragraph
<a> Defines a hyperlink
<img> Embeds an image
<ul>, <ol> Defines unordered and ordered lists
<table> Defines a table
<form> Defines an HTML form for user input
<input> Defines an input field
<div> Defines a division or a section
<span> Defines a section in a line
<button> Represents a clickable button
<!DOCTYPE> Defines the document type and version of HTML
<meta> Provides metadata about the HTML document
<link> Used to link external resources like CSS files
<script> Used to embed or refer to executable script
<iframe> Embeds another HTML page within the current page
